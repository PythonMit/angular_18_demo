=================================
DATA BINDING

1) one way binding

Interpolation          // {{}}  ,from ts to html (component file to view )
Property Binding      //  []   ,from ts to html (component file to view )
Event Binding        //   ()   ,from html to ts ( view file to component )

2) two way binding
Two-way Data Binding //   using ngModel  [(ngModel)]

3) using signal    
	define variable:: name:signal("name");
	set the value to variable :: this.name.set("name");
	
if you use [(ngmodel)] we need to import [FormsModule] in particular component.
